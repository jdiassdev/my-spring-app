# Spring – Fundamentos

Projeto de estudo para entender **como o Spring funciona na prática**.

## Objetivo

* Compreender a **estrutura básica** do Spring
* Entender **Controller, Service e Beans**
* Aprender **injeção de dependência** e **IoC**

## Conceitos

* `@Controller / @RestController`
* `@Service`
* `@Autowired`
* `@Bean`
* `entre outros`
* Spring gerencia o ciclo de vida dos objetos

## Estrutura

* Controller → entrada da aplicação
* Service → regra de negócio
* Repository → acesso a dados (quando necessário)

Projeto educacional focado em **fundamentos**.
